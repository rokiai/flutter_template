# 依赖说明（精简模板）

当前模板刻意去掉 Auth / Supabase / IAP / Drift / 音频等业务依赖，只保留脚手架。

## 保留

| 包 | 用途 |
|----|------|
| `flutter_riverpod` / codegen | 状态管理 |
| `go_router` / builder | 路由 |
| `dio` + `talker_dio_logger` | HTTP |
| `shared_preferences` | 主题 / 语言持久化 |
| `envied` | 环境变量 |
| `freezed` / `json_serializable` | 模型 |
| `sentry_flutter` | 可选崩溃监控 |
| `connectivity_plus` | 离线提示 |
| UI：`google_fonts` / `hugeicons` / `lottie` / `flutter_svg` | — |

## 环境

```
API_BASE_URL=https://jsonplaceholder.typicode.com
SENTRY_DSN=
```

`--dart-define=FLAVOR=dev|staging|prod`
